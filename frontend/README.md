# weeapps-client

Frontend for the weeapps playground written in Vue.js

## Project setup

```sh
yarn
```

### Compiles and hot-reloads for development

```sh
export VUE_APP_WEBSOCKET_SERVER_URL=wss://<your-weeapps-backend>
yarn serve
```

### Compiles and minifies for production

```sh
yarn build
```

### Code formatter

```sh
yarn prettier
```
